This section clearly distinguishes setup mechanisms across proof systems:

- Groth16: per-circuit setup
- UltraPlonk / UltraHonk (Noir): universal SRS + circuit-specific vk
- zkVM (Risc0 / SP1): transparent (no trusted setup)
