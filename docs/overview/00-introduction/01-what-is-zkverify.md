(A Verification Layer: it verifies proofs, it does not generate proofs)

This section explains zkVerify’s role in developer-friendly terms:
zkVerify is infrastructure dedicated to verifying zero-knowledge proofs.
It does not help you generate proofs. Instead, it verifies proofs generated elsewhere and turns the verification result into a trusted, reusable output.
