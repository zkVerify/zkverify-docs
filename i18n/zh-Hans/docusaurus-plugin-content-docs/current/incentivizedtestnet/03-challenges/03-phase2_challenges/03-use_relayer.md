### Overview

Kurier 让你在 zkVerify 上便捷验证零知识证明。本挑战邀请你体验其功能、提交真实证明，并反馈改进建议。

这是激励测试网 Phase 2 的技术挑战。完成任务可获取积分、提升排名。

**起步：**阅读 [Kurier 文档](https://docs.zkverify.io/overview/getting-started/kurier) 了解系统。

### Tasks & Rewards

#### 1. Submit Valid Proofs at Scale Using the Kurier API

- Kurier 使用量通过 API key 跟踪。需向团队或在 Discord 创建工单申请专属 API key。
- 重度用户可在 Discord 申请提升限速，需说明使用场景/应用，我们将据此调整。

<i>要获得积分，需要创建新电路、注册 vkey，并提交唯一的证明。支持的证明类型：Groth16、UltraHonk、UltraPlonk、Risc0、Plonky2、SP1、EZKL。</i><br/>

<br />

积分按合格用例的有效提交量计算。提交文档示例证明<b><u>不</u></b>计分。

<br />

| # of Unique Proofs Submitted | Points |
| ---------------------------- | ------ |
| 1-100                        | 5      |
| 101-500                      | 10     |
| 501-1000                     | 25     |
| 1000+                        | 60     |

#### 2. Provide Feedback on Kurier

通过反馈或报 bug 改进工具：

- Complete the [Kurier Feedback Form](https://forms.gle/PVjhLkDt2TbgmspGA)
- 报告未知 bug（含详细信息和截图）：+50 分

### How to Claim Points

提交证明或反馈后：

1. Fill out the [submission form](https://forms.gle/PVjhLkDt2TbgmspGA)
2. 提供 <b>EVM 地址</b> 及通过 Kurier 提交的 <b>API key、Job ID 或交易哈希</b>
3. zkVerify 团队会核验并分配积分

请始终使用分配的 API key，便于准确跟踪。
