### Overview

Kurier API 让你在 zkVerify 上轻松验证零知识证明。本挑战邀请你体验功能、提交真实证明并反馈改进意见。

完成下方任务可获取激励积分、提升排名。

**起步：**阅读 [Kurier 文档](https://docs.zkverify.io/overview/getting-started/kurier) 了解系统。

### Tasks & Rewards

#### 1. Submit Valid Proofs at Scale Using the Kurier API - Mainnet

- Kurier 使用量通过主网 API key 跟踪。需在 https://kurier.xyz 创建 API key。
- 重度用户可在 Discord 联系或填写 [表单](https://forms.gle/8GLNhADGoxkvCdTu7) 申请提升限速，需说明使用场景/应用。

<i>要获取积分，需要创建新电路、注册 vkey 并提交唯一证明。支持的证明类型：Groth16、UltraHonk、UltraPlonk、Risc0、Plonky2、SP1、Fflonk。</i><br/>

<br />

积分按合格用例的有效提交量计算，提交文档示例<b><u>不</u></b>计分。**需提供已集成 Kurier 的 zk-app 链接才符合资格。**

<br />

| # of Unique Proofs Submitted | Points |
| ---------------------------- | ------ |
| 1-100                        | 15     |
| 101-500                      | 30     |
| 501-1000                     | 60     |
| 1000+                        | 100    |

#### 2. Provide Feedback on Kurier

提交反馈或报 bug 改进工具：

- Complete the [Kurier Feedback Form](https://forms.gle/79nLKD4gDd2ndtar7)
- 报告未知 bug（含详细信息；截图可发送至 kurier-support@horizenlabs.io）：+50 分

### How to Claim Points

提交证明或反馈后：

1. Fill out the [submission form](https://forms.gle/hVutiC2Syb16bDg37)
2. 提供 <b>EVM 钱包地址</b>，以及通过 Kurier 提交的 <b>邮箱、应用链接</b>
3. zkVerify 团队将核验并分配积分

请始终使用你的 API key，确保活动可被准确追踪。
