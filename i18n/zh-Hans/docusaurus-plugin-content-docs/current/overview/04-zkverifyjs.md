按照下述说明，使用 [zkverifyjs](https://www.npmjs.com/package/zkverifyjs) npm 包提交 proof。

开始前，请确认本机已安装 Node.js。你可以从 [Node.js 官方网站](https://nodejs.org/) 下载并安装。

运行以下命令，确认 Node.js 与 npm 已正确安装。正常情况下会输出它们的版本号：

```shell
node -v
npm -v
```
